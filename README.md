#### 1. Clone the Repository

```bash

```

#### 2. Backend (Node.js)

```bash
cd backend-node
npm install
cp .env.example .env   # Add your DB and secret config

```

#### 3. Backend (Python - AI & Sentiment)

```bash
cd ../backend-python
pip install -r requirements.txt
python app.py
```

#### 4. Frontend (React)

```bash
cd ../frontend-react
npm install
npm run dev
```
